一、登入與客戶資料管理

  1. 輸入顧問姓名或 Email 與密碼登入。
  2. 一般顧問只會看到自己建立或負責的客戶。
  3. 完成客戶資料輸入後，請到「11.客戶管理」點選「將目前內容存入」。
  4. 下次服務同一位客戶時，可在客戶管理按「載入」繼續修改。
最佳做法：每次會談結束後立即存檔，並在 Memo 補上本次客戶重點與下一步行動。

二、建議操作流程

步驟操作頁面目的
10.基本資料確認客戶、配偶、家庭成員與資產基礎。
21.家庭保障估算主要收入者離開後家庭責任金額。
32.工作收入檢視失能、傷病或收入中斷時的現金流缺口。
43.醫療品質確認病房費、日額、雜費與重疾準備。
54.退休養老把退休年齡、壽命、生活費轉成退休需求。
66.需求缺口、9.財務報表用圖表與年度表格說明優先順序。

三、各頁填寫重點

  • 基本資料：先填姓名、生日、收入、月儲蓄；資產可先用概算。
  • 家庭保障：先填每月責任金額與年期，再填本人／配偶負擔比例。
  • 工作收入：以「每月需要多少錢」開始訪談，再轉成一次性準備金。
  • 醫療品質：A病房費、B日額用「元」；C雜費、D重疾用「萬元」。
  • 退休養老：先填退休年齡與預計壽命，再填基本與優質生活費。
  • 遺產稅：僅作初步估算，正式申報仍須依國稅局與專業人士確認。

四、報告列印與交付

  1. 點選右上角「列印」。
  2. 勾選封面、摘要、基本資料與需要的分析頁。
  3. 系統會依勾選頁面產生 A4 報告與頁碼。
  4. 建議交付順序：封面 → 摘要 → 基本資料 → 家庭保障 → 醫療／退休 → 財務報表 → Memo。
若圖表未出現，請先切換到相關頁面或重新點選列印；系統會在列印前將圖表轉為圖片。

五、Notion 匯出

  1. 完成客戶資料後，點選「Notion CSV」。
  2. 到 Notion 選擇 Import / 匯入 CSV。
  3. Notion 會依欄位建立客戶資料庫。
  4. 建議資料庫欄位包含客戶姓名、顧問、總缺口、退休缺口、下一步行動。
